## 2. 安装

1. 安装Homebrew：

	> $ /usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
	>
	> 测试：brewdoctor
	
2. 更新Homebrew（可选）：

	> brew update
	
3. 安装python3

	> brew install python3
	> 
	> 测试：which -a python 检测python共存情况。
	
	
4. 设定PATH路径信息：

	> sudo emacs /etc/paths
